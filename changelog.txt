v1.2
    + добавлена обработка наиболее вероятных ошибок и отображение их пользователю.

v1.1:  
    + добавлена группировка файлов по маске имени. Файлы с именами:
        messages.properties
        messages_xx.properties
        messages_yy.properties
    будут рассматриваться как файлы одного бандла и их содержимое будет сравниться только между собой.
    + добавлен вывод списка проигнорированных файлов. Игнор-лист выводится в окно сравнения файлов.
    
    = текстовый поле выбора каталога заменено на directory chooser;
    = разделена логика анализа файлов и сравнения их содержимого. Анализ производится и отображается сразу по выбору каталога, сравнение содержимого файлов запускается нажатием на кнопку "Compare";
