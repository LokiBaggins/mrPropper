Вас приветствует утилита обработки тектовых бандлов Mr. Propper!

Назначение:
1. Упростить жизнь человку за счет работящего, но бездушного железа;
2. Ускорить рутинные процессы сравнения файлов локализаций и поиска дублирующихся ключей в них;
3. Уменьшить количество ошибок, связаннх с текстами локализаций.

Возможности:
1. Обнаружение дублирующихся ключей в каждом файле.
    Результат выводится для каждого файла отдельно, с указанием:
        имя ключа-дубликата, 
        связанные с ним тексты + номера строк файла (где находятся эти значения).
        
2. Сравнение всех указанных файлов на предмет недостающих в них переводов.  
    Файлы группируются по именам

    Файлы с именами:
        messages.properties
        messages_xx.properties
        messages_yy.properties
        ValidationMessages.properties
        ValidationMessages_xx.properties
        
    будут разбиты на 2 группы (messages... и ValidationMessages...) и их содержимое будет сравниться только с файлами "своей" группы.
    Результ выводится в виде списка файлов и каких переводов в недостает в каждом из них.
       
Утилита исключительно аналитическая, т.е. никаких измений в анализируемых файлах не производится. 
       

!!! ВНИМАНИЕ !!!
Приложение обрабатывает исключительно файлы типа "*.properties". Все прочие игнорируются.
Список проигнорированных файлов выводится в окно сравнения.

Как пользоваться:

1. Скопируйте необходимые файлы .properties в папку "messages" (предпочтительно в альфа-версии)
    или укажите путь к необходимому каталогу.
2. Запустите приложение mrPropper-{version}.jar
3. Убедитесь, что путь в инпуте "Directory, where the files live:" указывает на выбранный Вами каталог.
4. Нажмите кнопку "Compare". 
Готово!

Краткая статистика отображается в табличном виде.
Чтобы увидеть более подробную информацию об отдельном файле и дубликатах, кликните по нужному файлу в таблице.

Результат сравнения файлов между собой отображается в текстово поле внизу экрана приложения.

Приятного пользования!
